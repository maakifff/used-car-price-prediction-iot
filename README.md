# Used Car Price Prediction Project

## 1. Problem Statement
We want to **predict selling price** for **used car market listings**, because it impacts **profit margin**.

We will use data from **historical sales datasets** at **individual car level**. Success looks like **95% accuracy**.

**Key Constraints:** Latency.

## 2. Project Description
This project aims to develop a machine learning model to estimate used car prices effectively. The solution will be deployed on an AWS EC2 instance as required by the course guidelines.
